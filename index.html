Example Project 01

Visual system for a temporary exhibition

Client: Dummy Studio
Category: Graphic Design
Design: Seesun Sohot

This dummy project explores a quiet visual rhythm through oversized type, strict margins, and a compact information system. The work includes printed matter, exhibition captions, digital announcements, and lettering tests.

Materials are arranged with a simple black-and-white structure so every project can be read quickly while still leaving room for detail, scale, and texture.
